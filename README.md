# ControlME — Personal Finance Tracker

**ControlME** is a personal finance tracking web application built with **React (JSX)** and **Vite**.  
It helps users monitor their income, expenses, balance, and monthly spending limits through a sleek **dark-themed UI** with glass-style cards and simple analytics.

## Features:

- Add and view **income and expense operations**
- Automatic calculation of **total balance**
- Track **monthly spending limits**
- Simple **analytics dashboard** with basic charts
- Clean and modern **dark green theme**
- Responsive layout for desktop screens  
(*Design inspired by modern finance dashboards and by lack of knowledge code in only one file app.jsx*)

## Built With:

- React (JSX)
- Vite (development and bundling)
- Basic chart component for analytics
- Custom CSS (dark theme)

## How to run site:

### Prerequisites

Install Node.js (v14+) and npm/yarn.

### Installation

1. Clone the repo:
```
git clone https://github.com/KYKAGM/Finance-Tracker
```
Go to project directory:
```
cd Where project located/Finance-Tracker
```
Install dependencies:
```
npm install
```
Run the development server:
```
npm run dev
```
It will open browser immediately (if its not open ur local host url)
```
http://localhost:5173
```
